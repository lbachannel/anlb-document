CSS Units (đơn vị trong css)

  • Absolute units (tuyệt đối)
    • px (pixel)
  • Relative units (tương đối)
    • %
    • vw, vh
    • em
    • rem

đơn vị %

<!-- 
    LƯU Ý: trường hợp đặc biệt
    khi box con padding-top. 
    KQ: padding-top: 50% là 1/2 width của cha
-->
<div class="box-parent">
    <div class="box-child" style="padding-top: 50%">
    </div>
</div>

đơn vị rem

  • rem
    • dựa trên font-size của thẻ html (thường là 16px mặc định)
    • 100% = 16px => 10px = (10 * 100) / 16 => 62.5% làm mặc định font-size cho thẻ html

đơn vị vw, vh

  • vw
    • (viewport width): 1vw = 1% chiều rộng màn hình.
  • vh
    • (viewport height): 1vh = 1% chiều cao màn hình.
