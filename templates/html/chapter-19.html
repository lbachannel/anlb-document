CSS colors

  • Names
    • 140 colors: black, gray, green...
  • rgb(r, g, b);
    • 3 thông số này sẽ chạy từ 0 - 255 cực đại
    • rgb(r g b / alpha)
    • rgb(r, g, b, 50%)
  • rgba(r, g, b, a);
    • a: alpha (độ trong suốt)
    • 3 thông số này sẽ chạy từ 0 - 255, thông số cuối 0 - 1

    • hệ thập lục phân: 0123456789abcdef cực đại
