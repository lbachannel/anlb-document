Nối chuỗi và nội suy

Tạo chuỗi với nháy đơn

Đặc điểm: cách truyền thống.

Ứng dụng: không cần xuống dòng. Rất hữu ích khi chuỗi chứa các dấu nháy kép bên trong.

Ví dụ: : 'Hello, this is "JavaScript"!'.

Tạo chuỗi với nháy kép

Đặc điểm: cách truyền thống, khi chuỗi chứa dấu nháy đơn.

Ứng dụng: Hữu ích khi muốn bao gồm dấu nháy đơn trong chuỗi mà không cần dùng ký tự thoát (Escape character).

Ví dụ: : "Hello, this is 'JavaScript'!".

Tạo chuỗi với backticks

Đặc điểm: Hỗ trợ nội suy chuỗi (cho phép chèn biến hoặc biểu thức vào chuỗi), và cho phép xuống dòng trực tiếp trong chuỗi.

Ứng dụng: Cực kỳ hữu ích khi cần kết hợp chuỗi với biến hoặc khi cần tạo chuỗi nhiều dòng.

Ví dụ: : 

let name = "JavaScript";

`Xin chào, đây là ${name}!
Chúng ta có thể xuống dòng trực tiếp.`;

Sử dụng ${name} bên trong backticks được gọi là “Nội suy”

Nối chuỗi

Toán tử nối chuỗi

let greeting = "Hello, " + "my name is " + "An."; // Output: Hello, my name is An.
'Hello' + ' ' + 'An'; // Output: Hello An

Lưu ý khi sử dụng

5 + 4 + '6' // Output: "96" chứ không phải 546. Vì phép cộng sẽ thực hiện trước, sau đó nối với chuỗi "6"
"1" + 2 + 3 // Output: "123". Vì "3" là chuỗi khiến các số tiếp theo được chuyển thành chuỗi và nối lại

Cách làm sai

let a = 10;
let b = 20;
let message = 'Tổng của ' + a + ' và ' + b + ' là: ' + a + b;
console.log(message); // Tổng của 10 và 20 là: 1020

Cách đúng (cách 1)

let a = 10;
let b = 20;
let message = 'Tổng của ' + a + ' và ' + b + ' là: ' + (a + b);
        
console.log(message); // Tổng của 10 và 20 là: 30

Cách đúng (cách 2)

let a = 10;
let b = 20;
let total = a + b; // Thực hiện phép cộng trước
let message = 'Tổng của ' + a + ' và ' + b + ' là: ' + total;
        
console.log(message); // Tổng của 10 và 20 là: 30

Cách đúng (cách 3 - nên dùng cách này)

let a = 10;
let b = 20;
let total = a + b; // Thực hiện phép cộng trước
let message = `Tổng của ${a} và ${b} là: ${total}`;
        
console.log(message); // Tổng của 10 và 20 là: 30

Template literals

Template literals là gì

Là tính năng mới được giới thiệu trong ECMAScript 2015 (còn được gọi là ES6).

Template Literals sử dụng backticks (`) để định nghĩa chuỗi thay vì dấu nháy đơn hoặc nháy kép.

Điểm đặc biệt của Template Literals là khả năng thực hiện nội suy - tức là chèn biến hoặc biểu thức trực tiếp vào trong chuỗi.

Cú pháp ${expression}.

Ngoài ra, Template Literals còn hỗ trợ việc tạo chuỗi đa dòng mà không cần sử dụng các ký tự xuống dòng như \n.

“Template literals” còn được gọi là “Template strings”. Cả hai cách gọi đều nói tới việc sử dụng backticks để tạo chuỗi với đặc điểm là cho phép nội suy - tức là chèn biến hoặc biểu thức trực tiếp vào trong chuỗi.
