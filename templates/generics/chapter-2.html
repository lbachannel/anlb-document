Bounded type parameters

public class Myposition <T extends Number> {
    private T x;
    private T y;

    public T getX() {
        return x;
    }

    public void setX(T x) {
        this.x = x;
    }

    public T getY() {
        return y;
    }

    public void setY(T y) {
        this.y = y;
    }
}

Tham số <> phải là lớp con của Number 

public static void main(String[] args) {
    MyPosition<Integer> pos = new MyPosition();
    pos.setX(5);
    pos.setY(6);
}

Đây gọi là kỹ thuật ràng buộc kiểu Generics (Bounded type parameters)
