Multiple types multiple bounds

public class Demo {
    public static void main(String[] args) {
        maxValue(2, 5);
    }

    // multiple types
    public static <T extends Integer, T1 extends Integer> void maxValue(T x, T1 y) {
        if (x > y) {
            System.out.println(x);
        } else {
            System.out.println(y);
        }
    }

    // multiple bounds 
    // T kế thừa number & T phải thực hiện (implement) interface Comparable.
    public static <T extends Number & Comparable<T>> void maxValue(T x, T y) {
        if (x.compareTo(y) > 0) {
            System.out.println(x);
        } else {
            System.out.println(y);
        }
    }
}

Trường hợp trên bắt buộc phải extends Integer vì thực hiện so sánh 2 vế

Nếu extends Number, Float, Double, Byte. Sẽ nhận thông báo lỗi khi biên dịch chương trình. Hoặc nhận kết quả sai khi thực thi chương trình.
