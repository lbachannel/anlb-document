Install SQL Server 2019

Step 1:

Link download: click me

Download version Express (SQL2019-SSEI-Expr.exe)

Step 2:

run => select an installation type: Download Media

Which package would you like to download? => Express Advanced

When download successful! => clone (SQLEXPRADV_x64_ENU.exe - server)

Step 3:

Vào trang chủ Microsoft tìm đến mục tool - Download SQL Server Management Studio (SSMS) => Click vào

Tìm tiêu đề Download SSMS - Download SQL Server Management Studio (SSMS) 20.2

SSMS-Setup-ENU.exe (client)

Step 4:

Tiến hành cài file SQLEXPRADV_x64_ENU.exe - server

Choose Directory For Extracted Files => Click oki

Hộp thoại: SQL Server Installation Center

Nếu cài mới hoàn toàn thì chọn: New SQL Server start alone installation or add features to an existing installation

Nếu update thì chọn Upgrade from a previous version of SQL Server

Step 5:

Hộp thoại SQL Server 2019 setup: Uncheck Machine Learning Services and Language => Next, next...done and close

Step 6:

Tiến hành install file đã download ở step 3 (SSMS-Setup-ENU.exe (client)). Đây là client tool sẽ giao tiếp với server
