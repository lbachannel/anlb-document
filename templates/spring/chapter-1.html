JSON

Json: viết tắt của JavaScript Object Notation

JSON không thuộc riêng về JavaScript. Thực tế, JSON là một định dạng dữ liệu độc lập với ngôn ngữ.

JSON là một định dạng văn bản thường được dùng để truyền dữ liệu giữa các hệ thống, đặc biệt là khi giao tiếp giữa client (ví dụ: ứng dụng di động, trình duyệt) và server thông qua API

Ví dụ này là 1 chuỗi JSON 

'{"name":"John", "age":30, "car":null}'

JSON -> Object 

JSON.parse()

const obj = JSON.parse('{"name":"John", "age":30, "city":"New York"}');

Object -> JSON 

JSON.stringify()

const obj = {name: "John", age: 30, city: "New York"};
const myJSON = JSON.stringify(obj);

Nested JSON object là một đối tượng JSON chứa các đối tượng JSON khác hoặc mảng JSON bên trong nó. 

{
    "name": "Lai Binh An",
    "age": 18,
    "skills": ["Java", "JavaScript", "SQL"],
    "job": {
        "title": "Software Engineer",
        "experience": {
            "years": 1,
            "projects": ["Project A", "Project B"]
        }
    }
}
