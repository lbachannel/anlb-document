JSX - Fragment

Fragment: nếu không có fragment sẽ lỗi, vì chỉ tối đa 1 phần tử. Chúng ta bọc lại các elements với fragment vẫn đảm bảo tối đa 1 phần tử 

const MyComponent = () => {
    return (
        <>
            <div>div 1</div>
            <div>div 2</div>
            <div>div 3</div>
        </>
    );
}
