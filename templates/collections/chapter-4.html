Queue Interface

Phần tử được đưa vào từ cuối (rear), và lấy ra từ đầu (head)

Còn được gọi là cấu trúc dữ liệu first-in first-out

Có 2 lớp hiện thực

LinkedList (First-in First-out FIFO)

PriorityQueue

LinkedList

public static void main(String[] args) {
    Queue queue = new LinkedList();
    queue.add("first");
    queue.add("second");
    queue.add("third");

    Iterator it = queue.iterator();
    while(it.hasNext()) {
        System.out.println(it.next());
    }
}

Lấy phần tử đầu nhưng không xóa phần tử 

Object element = queue.peek(); // first

Lấy phần tử đầu và đồng thời xóa phần tử này 

Object element = queue.poll(); // first

Delete method (return boolean) 

boolean isDelete = queue.remove("second"); // true
