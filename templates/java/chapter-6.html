Primitive Array

Trong Java, cấm tuyệt đối chỉ định kích thước mảng trong lệnh khai báo 

double vol[365]; // => error
double vol[365] = {0, 5, 10, 15, 20, 25, 30}; // => error

Cú pháp đúng: 

double v[] = {0, 5, 10, 15, 20, 25, 30}; // new ngầm, implicit

hoặc 

double arr[] = new double[365]; // new rõ ràng, explicit
double array[] = new double[]{5, 10, 15, 20, 25};

Không nguy hiểm khi loop. Vì primitive nếu chưa gán thì default là 0 thì in ra 0.

Object Array

Student s[] = new Student[50];
s[0] = new Student("se001", "ahihi", 1999, 9.0);
s[1] = new Student("se002", "ahoho", 1999, 8.5);
s[2] = new Student("se003", "ahehe", 1999, 7);

for (int i = 0; i < 5; i++) {
    s[i].showProfile();
}

// => when s[3] jvm will kill the app. 
// Because JVM will send you Null pointer Exception. 
// Since the variable has not been assigned, the default value is zero.

Nguyên tắc khi chơi với mảng Object

Không được loop hết, mà chỉ loop đến chỗ nào đã gán. Vì phần còn lại trỏ null, default = 0

Chơi mảng obj là cần biến count đếm xem đã có bao nhiêu biến được gán value

Sort Object Array

public static void sortObjectArray() {
    Student[] arr = new Student[2];
    arr[0] = new Student("se001", "ahihi", 9.0);
    arr[1] = new Student("se002", "ahoho", 5.0);
    
    if (arr[0].getGPA() > arr[1].getGPA()) {
        Student tmp = arr[0];
        arr[0] = arr[1];
        arr[1] = tmp;
    }
}

Bubble Sort 

public static void sortObjectArray() {
    int arr[] = {1000, -1000, 5, 15, -20, 3, 7, 33};
    
    for (int i = 0; i < arr.length - 1; i++) {
        for (int j = i + 1; j < arr.length; j++) {
            if (arr[i] > arr[j]) {
                int tmp = arr[i];
                arr[i] = arr[j];
                arr[j] = tmp; 
            }
        }
    }
}
