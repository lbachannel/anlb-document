Kiểu dữ liệu & biến

Biến là gì?

Biến là 1 vùng ram được đặt tên, chiếm 1 số byte nhất định tùy loại.

Dữ liệu, thông tin mà nó sẽ chứa bên trong

Vùng ram dùng để lưu trữ dữ liệu thông tin và xử lý.

Primitive Data Type

Có 8 loại dữ liệu nguyên thủy

Data type Số byte Số bit
byte 1 byte 8
short 2 byte 16
int 4 byte 32
long 8 byte 64
float 4 byte 32
double 8 byte 64
boolean không chỉ định rõ kích thước, hoặc 1 bit false/true
char 2 byte 16

Integer

int age = 20;
long m = 3000000000; // error
long m = 3000000000L; // correct

Mọi con số, kí tự xuất hiện trong code được gọi là literal value

Số nguyên xuất hiện trong code mặc định là int.

Double

Java ưu tiên double hơn float

float vat = 0.1; // error
float vat = 0.1F; // correct

Characters

char 2 byte, lưu bảng mã Unicode

char m = '$';

Boolean

boolean marriedStatus = false;
if (marriedStatus == true) {
    System.out.println("Đánh đồn có địch mới vui");
} else {
    System.out.println("Vườn hồng có lối nhưng chưa ai vào");
