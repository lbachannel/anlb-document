Functional Interface

A Functional Interface in Java is an interface that has exactly one abstract method. It can have other non-abstract methods, such as default or static methods, but only one method can remain abstract. 

@FunctionalInterface
interface LearnEnglish {
    void speakEnglish(String content);
    default void write() {...};
    public static void read() {...};
}

Traditional 

LearnEnglish learn = new LearnEnglish() {
    @Override
    public void speakEnglish(String content) {
        System.out.println(content);
    }
    learn.speakEnglish("I don't know what 7 you say");
}

Lambda 

LearnEnglish learn = content -> System.out.println(content);
learn.speakEnglish("I don't know what 7 you say");
